# Features

Every user-facing feature of PowerAim, grouped by what it does. Pages are written so you can skim the headings — each one ends with **Tips**, **Configuration options**, and a link to relevant **Troubleshooting**.

## Aim & targeting

- **[Aim Assist]({{ '/features/aim-assist' | relative_url }})** — the core aim feature, FOV, sensitivity, EMA smoothening, predictions
- **[Triggers]({{ '/features/triggers' | relative_url }})** — multi-trigger AutoTrigger with charge mode and AND/OR operators
- **[Anti-Recoil]({{ '/features/anti-recoil' | relative_url }})** — three independent anti-recoil modes
- **[Recoil Patterns]({{ '/features/recoil-patterns' | relative_url }})** — record + play back per-gun drift curves
- **[Detection Masks]({{ '/features/detection-masks' | relative_url }})** — exclude HUD regions from detection
- **[Calibration Wizard]({{ '/features/calibration-wizard' | relative_url }})** — automatic sensitivity tuning

## Overlays & HUD

- **[Crosshair Overlay]({{ '/features/crosshair-overlay' | relative_url }})** — custom topmost crosshair
- **[Debug Overlay]({{ '/features/debug-overlay' | relative_url }})** — live stats panel
- **[FOV Overlay]({{ '/features/fov-overlay' | relative_url }})** — FOV circle + dynamic FOV on hotkey
- **[Session Stats]({{ '/features/session-stats' | relative_url }})** — FPS, inference time, detection count, shots fired

## Capture & data

- **[Replay Buffer]({{ '/features/replay-buffer' | relative_url }})** — rolling JPEG buffer of the last N seconds
- **[OCR]({{ '/features/ocr' | relative_url }})** — Tesseract-based HUD reader for health / ammo

## Controller & input

- **[Controller Mapping]({{ '/features/controller-mapping' | relative_url }})** — reWASD-style KB↔Pad remapping engine
- **[Gamepad Aim]({{ '/features/gamepad-aim' | relative_url }})** — use the virtual right stick instead of the mouse
- **[Hidden Controllers]({{ '/features/hidden-controllers' | relative_url }})** — HidHide cloak + device disable
- **[Movement Methods]({{ '/features/mouse-input-methods' | relative_url }})** — SendInput, MouseEvent, LGHub, Razer, ddxoft, Gamepad

## AI / automation

- **[AutoPlay]({{ '/features/autoplay' | relative_url }})** — Ollama-driven LLM that plays the game
