# Getting Started

This section walks you from "I just downloaded PowerAim" to "the crosshair is moving by itself in-game." It takes about 10 minutes from a clean machine.

There are four pages:

1. **[System Requirements]({{ '/getting-started/system-requirements' | relative_url }})** — hardware, OS, and what optional drivers do.
2. **[Installation]({{ '/getting-started/installation' | relative_url }})** — picking the right build (DirectML vs. CUDA), installing prerequisites, first launch.
3. **[Your First Aim]({{ '/getting-started/first-aim' | relative_url }})** — load a model, pick a key, and see PowerAim follow a target.
4. **[Setup Best Practices]({{ '/getting-started/setup-best-practices' | relative_url }})** — once it works, get it into a *good* state: setup order, per-game profiles, GPU selection, anti-detection knobs, common pitfalls.

{: .tip }
If you only have 5 minutes, skip ahead to [Your First Aim]({{ '/getting-started/first-aim' | relative_url }}) — it tells you the minimum number of clicks to see PowerAim work.

After the basics, you'll probably want to read:

- **[Aim Assist]({{ '/features/aim-assist' | relative_url }})** — every knob in the AimConfig card explained
- **[Triggers]({{ '/features/triggers' | relative_url }})** — the multi-trigger / charge-mode system
- **[Models]({{ '/models/using-models' | relative_url }})** — switching between the bundled models or downloading new ones
- **[Calibration Wizard]({{ '/features/calibration-wizard' | relative_url }})** — let PowerAim measure your in-game sensitivity for you
